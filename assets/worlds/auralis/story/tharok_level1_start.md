# Tharok Level 1 - Start
Mit dem Sporenkompass im Handschuh erreicht Bob die Felsguertel von Tharok, wo Alienbohrer den Himmel spalten. Auf Tharok ist das Ziel in diesem Einsatz, einen eingestuerzten Minenschacht raeumen. Bob unterstuetzt Minenleute und stellt den Resonanzkern wieder her.
Die Zone ist voll von Panzerkrabbler und bohrende Xenodrachen. Bob merkt, dass jede gewonnene Minute die Chance erhoeht, PlasmaBetty lebend zu finden.
