# Solkhar Level 6 - Start
Auf Solkhar enden alle Umwege; Bob stuermt direkt auf die Festung der Entfuehrer zu. Auf Solkhar ist das Ziel in diesem Einsatz, ein Waffenlager der Aliens sprengen. Bob stuermt die Festung der Entfuehrer, um PlasmaBetty zu retten.
Die Zone ist voll von Sandlaeufer, glutschwarme Insekten und Elitealiens. Bob merkt, dass jede gewonnene Minute die Chance erhoeht, PlasmaBetty lebend zu finden.
